Modelos y cuando usarlos
Tipo Problema_tipico Ventajas Cuando_usarlo
Regressión Valores numéricos Simplicidad Relaciones lineales
Árboles / Decision Tree Clasificación, regresión Interpretabilidad Datos tabulares
Ensambles Clasificación, regresión Precisión Alto rendimiento, Kaggle
Deep Learning Imágenes, texto, audio Modelos complejos Datos grandes y no estructurados
Reducción de Dim. Visualización, preprocesamiento Mejora eficiencia Datos con muchas variables
Bayesianos Clasificación rápida Velocidad Texto, spam detection
Regularización Evitar overfitting Generalización Modelos lineales con muchas variables
Instance-Based Clasificación Simple, no requiere entrenamiento Pocos datos y relaciones claras
Clustering Agrupamiento no supervisado Descubrir estructuras ocultas Segmentación sin etiquetas
Rule-Based Interpretabilidad Lógica clara Reglas conocidas, decisiones explicables
