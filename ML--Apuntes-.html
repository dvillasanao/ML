Hopfield Network

La Red de Hopfield es un tipo de red neuronal recurrente o red neuronal con memoria asociativa, propuesta por John Hopfield en 1982. La regla de aprendizaje m√°s com√∫n para establecer estos pesos es la regla de Hebb: si dos neuronas se activan juntas para un patr√≥n, el peso entre ellas se incrementa.
